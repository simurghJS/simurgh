کار با api

جهت راحتی در فراخوانی توابع api در محیط gilacejs ، کلاس request به صورت سراسری(global) در controller ها در دسترس است.

get

ارسال یک درخواست با متد get و نمایش مقدار param1 از نتیجه api

new request('path/to/api').get().then((response) => {

alert(response.param1);

});

post

این متد جهت ارسال اطلاعات به سمت سرور با متد post میباشد.

ارسال مقادیر name و age به سرور جهت آپدیت کاربر با id شماره ۱

new request("user/update/1").post({

name:'حسن',

age:27

}).then((response) => {

alert(response.param1);

});
