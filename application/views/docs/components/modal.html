پیکره بندی

gilacejs در زمان شروع به کار, یک نسخه از اپلیکیشن با تنظیماتی که در فایل app.js نوشته اید (شامل نام اپلیکیشن, آدرس api و ...) ایجاد می کند و تا پایان کار به عنوان مرجع از آن استفاده خواهد کرد.در این بخش کار با فایل app.js را یاد میگیرید

نصب و اجرا

برای نصب ابتدا GilaceJS را از لینک زیر دریافت نمایید.

https://gilace.ir/siyamak/gilacejs.git

.سپس جهت نصب نیازمندی ها، در پوشه پروڑه، دستور npm install را اجرا نمایید. پس از نصب برای اجرا، فایل index.html را در مرورگر باز نمایید.

شروع سریع

فایل app.js نقطه شروع کار می باشد. در این فایل شما میبایست مشخصات اپلیکیشن خود را تعریف نمایید. فایل کنونی حاوی کد های نمونه مربوط به این آموزش است. پس از بررسی مختصر این فایل گام بعدی برای ایجاد یک صفحه جدید تعریف یک route جدید است.route ها در مسیر config/routes.js قرار دارند. کد زیر را در انتهای این فایل کپی کنید

ایجاد route جدید

router.route('testURL', 'index').name('testNav');

اکنون شما یک route جدید ساخته اید که اگر کاربر آدرس testURL را در اپلیکیشن شما اجرا نماید, سیستم فایل index.js موجود در مسیر application/controllers را اجرا میکند. برای نهایی سازی این فایل را ایجاد نمایید و کد زیر را در آن کپی نمایید تا نتیجه را مشاهده کنید.

کد صفحه نمونه

import {BaseController} from '/gilace/gilace.js'

class Index extends BaseController{

start(navigation_data = {}) {

return 'hello world!';

}

}

export default Index;
