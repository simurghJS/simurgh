ساختار پوشه ها

پیش از شروع آموزش بهتر است با ساختار پوشه های gilacejs آشنا شوید, سعی ما بر اینست تا با توضیحی مختصر و کوتاه شما را با پوشه ها و کاربردشان آشنا کنیم

application

کل فایل های اجرایی اپلیکیشن شما در این پوشه قرار می گیرد.این فایل ها عبارتند از

  • controllers

    فایل های اجرایی js مربوط به ساخت صفحات

  • middleware

    فایل های js مربوط به middleware میبایست در این پوشه قرار گیرد

  • models

    مدل ها برای ایجاد خودکار controllers های عملیات crud میباشد

  • views

    فایل های html حاوی ui

assets

سایر فایل های مربوط به ui همانند تصاویر , فایل های css و ... در این پوشه قرار می گیرد. شما از طریق تابع assets میتوانید به این فایل ها دسترسی داشته باشید.

مسیر فایل های ui

let image_src = assets('img/example.png');

config

تنظیماتی شامل route ها و ... در این پوشه قرار میگیرد

gilace

فایل های js مربوط به gilacejs. اعمال تغییرات در این فایل ها میتواند موجب مختل شدن عملکرد اپلیکیشن شما شود, پس مراقب باشید
