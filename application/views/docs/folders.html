توابع کمکی

توابع کمکی شامل برخی از پرکاربرد ترین توابع مورد استفاده میباشد که میتوانند سرعت توسعه را افزایش دهند.در ادامه لیست توابع در دسترس در اختیارتان قرار خواهد گرفت

empty(object)

این تابع بررسی میکند شیئ ورودی خالیست یا خیر و true  | false برمیگرداند.

نمونه

let data = $('#target').data('navigate');

if (!empty(data)) {

//do stuff

}
