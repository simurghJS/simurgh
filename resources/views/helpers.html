ابزار های کمکی

ابزارهای کمکی شامل برخی از پرکاربرد ترین توابع مورد استفاده است, که میتوانند سرعت توسعه را افزایش دهند. همچنین جهت راحتی در فراخوانی توابع api در محیط {{env('app_name')}} ، کلاس request به صورت سراسری(global) در کامپوننت ها در دسترس است.

کار با api

ارسال درخواست با متد GET به آدرس example.com/path/to/api

                            
                               let response = await new request('path/to/api').get();

ارسال اطلاعات با متد POST/PUT/PATCH به آدرس example.com/path/to/api

                            
                                let data = {
                                                name:      'حسن',
                                                family:    'ترابی',
                                                age:        27,
                                                gender:    'male'
                                            }

                                //let response = await new request("path/to/api").path(data);
                                //let response = await new request("path/to/api").put(data);
                                let response = await new request("path/to/api").post(data);

                                if( response.status == 200 ){

                                    alert('success');

                                }

ارسال درخواست با متد DELETE به آدرس example.com/path/to/api

                            

                                let response = await new request("path/to/api").delete();
                                if( response.status == 200 ){

                                    alert('success');

                                }
