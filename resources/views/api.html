کار با api

جهت راحتی در فراخوانی توابع api در محیط gilacejs ، کلاس request به صورت سراسری(global) در controller ها در دسترس است.

get

ارسال درخواست با متد GET به آدرس example.com/path/to/api

                            
                               let response = await new request('path/to/api').get();

post

ارسال اطلاعات با متد POST به آدرس example.com/path/to/api

                            
                                let data = {
                                                name:      'حسن',
                                                family:    'ترابی',
                                                age:        27,
                                                gender:    'male'
                                            }

                                let response = await new request("path/to/api").post(data);
                                if( response.status == 200 ){

                                    alert('success');

                                }
