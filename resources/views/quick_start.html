شروع سریع

فریم ورکی بر پایه جاوااسکریپت برای توسعه آسان و سریع UI میباشد {{env('app_name')}}

دانلود و نصب

برای نصب ابتدا {{env('app_name')}} را از لینک زیر دریافت نمایید, و در پوشه ای با نام پروژه خود, فایل را از حالت فشرده خارج کنید. سپس جهت نصب سایر نیازمندی ها دستور npm install را اجرا نمایید. فایل نصبی شامل اپلیکیشن پایه است. دستور npm run build-web را اجرا کنید. پس از چند لحظه اپلیکیشن شما در پوشه dist/web ایجاد میشود. برای اجرای اپلیکیشن در local میبایست ابتدا با ابزار هایی همانند MAMP یک هاست ایجاد کنید(در طول این آموزش ما آدرس example.com را به عنوان آدرس هاست در نظر میگیریم) و مسیر آن را پوشه dist/web قرار دهید. اکنون میتوانید مرورگر را باز کنید و نتیجه را در آدرس example.com مشاهده نمایید

لینک دانلود http://arnahit.com/downloads/arnahitJS.zip

ساختار پوشه ها

پیش از شروع بهتر است با ساختار پیش فرض پوشه ها در {{env('app_name')}} آشنا شوید.

  • app

    فایل های اجرایی js مربوط به صفحات

  • resources

    کل فایلهای مربوط به ui در این پوشه قرار میگیرد

  • core

    فایل های مربوط به {{env('app_name')}}

  • dist

    فایلهای خروجی

آشنایی سریع

فایل app.js نقطه شروع کار میباشد. شما در این فایل تنظیمات اپلیکیشن خود برای اجرا در مرورگر را معرفی میکنید. برای شروع سریع شما نیاز دارید تا یک route به سیستم اضافه کنید.route ها نحوه پاسخ اپلیکیشن به url های درخواستی را مشخص میکنند. تغییرات زیر را در فایل app.js اعمال کنید و نتیجه را در مرورگر ببینید. برای اینکار آدرس example.com را در مرورگر باز کنید.

ایجاد یک route جدید
                            
                                // import Router class
                                import {Application, Router} from '/gilace/quick_start.js';

                                let myApp = new Application();

                                // add this code to app.js

                                myApp.registerRoutes(()=>{
                                    let router = new Router();
                                    router.route('',()=>{
                                        return 'hello world 1';
                                    });
                                });

                                //end
                                ...
                                export default myApp;

همچنین شما میتوانید یک controller بسازید و به جای پاسخ مستقیم از طریق router ، دستور اجرای controller را بدهید. برای این کار فایل quick_start.js را در پوشه پروڑه و مسیر application/controllers بسازید و کد زیر را در این فایل کپی کنید

کد پایه controller با نام index
                            
                                import {Component} from '/quick_start.js'

                                class Quick_start extends Component{
                                    start(navigation_data = {}) {
                                        return 'hello world!';
                                    }
                                }

                                export default Quick_start;

برای اجرای این فایل در ازاء اجرای آدرس example.com/#!/test میبایست یک route جدید ایجاد کنید. برای اینکار کد زیر را به فایل app.js اضافه کنید

                            
                                router.route('test','index');

برای اجرای یک controller تعریف route الزامی نیست. اگر controller متناظر با آدرس نوشته شده پس از /#!/ در url عینا در پوشه application/controllers موجود باشد شما نیازی به تعریف route ندارید. برای تست این موضوع کد پایه controller را در فایلی با نام test2.js در مسیر application/controllers/custom_url کپی کنید. در اینصورت اگر در مرورگر آدرس example.com/#!/custom_url را بازکنید میبایست نوشته hello world! را ببینید
